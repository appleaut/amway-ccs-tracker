# คู่มือการติดตั้ง — Amway CCS Tracker

โปรแกรมบันทึกผู้มุ่งหวัง / ลูกค้า / ดาวน์ไลน์ สำหรับ Windows

---

## 1. ความต้องการของระบบ

- **Windows 10 หรือ 11** (64‑บิต)
- พื้นที่ว่างประมาณ **20 MB**
- **ไม่ต้องใช้สิทธิ์ผู้ดูแล (admin)** — ติดตั้งในบัญชีผู้ใช้ของตัวเองได้เลย

> โปรแกรมหลักทำงานได้ทันทีโดยไม่ต้องติดตั้งอะไรเพิ่ม (ฟอนต์ไทยและฐานข้อมูลฝังมาในตัวแล้ว)

---

## 2. วิธีติดตั้ง

1. นำไฟล์ **`AmwayCCSTracker-Setup.exe`** มาไว้บนเครื่อง (เช่น โฟลเดอร์ Downloads)
2. **ดับเบิลคลิก** ที่ไฟล์เพื่อเริ่มติดตั้ง
3. **ถ้ามีหน้าจอสีฟ้า "Windows protected your PC"** ขึ้นมา (เป็นเรื่องปกติเพราะไฟล์ยังไม่ได้เซ็นรับรองดิจิทัล ไม่ใช่ไวรัส) ให้ทำดังนี้:
   - คลิก **"More info"** (ข้อมูลเพิ่มเติม)
   - แล้วคลิกปุ่ม **"Run anyway"** (เรียกใช้งานต่อไป)
4. อ่านหน้าข้อมูลเบื้องต้น แล้วกด **Next**
5. เลือกว่าจะสร้าง **ไอคอนบนเดสก์ท็อป (Desktop shortcut)** หรือไม่ (แนะนำให้ติ๊กไว้) แล้วกด **Next → Install**
6. เมื่อเสร็จ กด **Finish** (ติ๊ก "Launch Amway CCS Tracker" ไว้เพื่อเปิดโปรแกรมทันที)

โปรแกรมจะถูกติดตั้งไว้ที่:
```
C:\Users\<ชื่อผู้ใช้>\AppData\Local\Programs\AmwayCCSTracker
```
และจะมีรายการ **"Amway CCS Tracker"** อยู่ในเมนู Start

---

## 3. การเปิดใช้งาน

เปิดจาก **เมนู Start** หรือ **ไอคอนบนเดสก์ท็อป** ได้เลย — โปรแกรมพร้อมใช้งานทันที

ข้อมูลทั้งหมด (รายชื่อ, การประชุม, สิ่งที่ต้องทำ, สำรองจ่าย ฯลฯ) จะถูกบันทึกไว้ในเครื่องที่:
```
C:\Users\<ชื่อผู้ใช้>\AppData\Roaming\AmwayCCSTracker\data.db
```

---

## 4. การสำรองและย้ายข้อมูล

ไปที่เมนู **⚙ ตั้งค่า → ข้อมูลและการสำรอง**

- **สำรองข้อมูล (Backup):** บันทึกข้อมูลทั้งหมดเป็นไฟล์ `.db` ไฟล์เดียว เก็บไว้ที่ไหนก็ได้ (เช่น USB, Google Drive)
- **กู้คืนข้อมูล (Restore):** เลือกไฟล์สำรองที่เคยบันทึกไว้เพื่อนำข้อมูลกลับมา (ระบบจะสำรองข้อมูลปัจจุบันไว้ให้อัตโนมัติก่อนเขียนทับ)

> **ย้ายเครื่อง:** ทำ "สำรองข้อมูล" บนเครื่องเก่า → ติดตั้งโปรแกรมบนเครื่องใหม่ → "กู้คืนข้อมูล" จากไฟล์ที่สำรองไว้

---

## 5. ฟีเจอร์ "ดาวน์โหลดโปรโมชัน" (ไม่บังคับ)

ฟีเจอร์นี้ดึงรูปโปรโมชันประจำเดือนจากเว็บ amway.co.th — ต้องติดตั้งโปรแกรมเสริมก่อนใช้งาน:

1. **Google Chrome** (ติดตั้งตามปกติจาก google.com/chrome)
2. **Python 3** (จาก python.org — ตอนติดตั้งให้ติ๊ก "Add Python to PATH") แล้วเปิด Command Prompt พิมพ์:
   ```
   pip install playwright pillow
   ```

ถ้ายังไม่ได้ติดตั้งสองอย่างนี้ ส่วนอื่นของโปรแกรมยังใช้งานได้ตามปกติ — เฉพาะปุ่มดาวน์โหลดโปรโมชันเท่านั้นที่จะแจ้งว่ายังขาดโปรแกรมเสริม

---

## 6. การถอนการติดตั้ง

ไปที่ **Settings → Apps → Installed apps** (หรือ "Add or remove programs") → ค้นหา **"Amway CCS Tracker"** → **Uninstall**

> ✅ **ข้อมูลของคุณจะไม่ถูกลบ** — การถอนการติดตั้งจะลบเฉพาะตัวโปรแกรม แฟ้มข้อมูล (`data.db` และไฟล์สำรอง) ยังคงอยู่ หากติดตั้งใหม่ภายหลังจะเห็นข้อมูลเดิมครบ หากต้องการลบข้อมูลถาวร ให้ลบโฟลเดอร์ `AppData\Roaming\AmwayCCSTracker` ด้วยตนเอง

---

## 7. การแก้ปัญหาที่พบบ่อย

| อาการ | วิธีแก้ |
|--------|---------|
| หน้าจอ "Windows protected your PC" | คลิก **More info → Run anyway** (ปลอดภัย ไฟล์แค่ยังไม่ได้เซ็นรับรอง) |
| ขึ้น error `vcruntime140.dll is missing` (พบน้อยมากบนเครื่องใหม่เอี่ยม) | ติดตั้ง **Microsoft Visual C++ Redistributable (x64)** ฟรีจากเว็บ Microsoft แล้วเปิดโปรแกรมใหม่ |
| โปรแกรมป้องกันไวรัสเตือน/บล็อก | เป็น false positive เพราะไฟล์ยังไม่ได้เซ็น — อนุญาต (allow) ไฟล์ไว้ |
| ปุ่มดาวน์โหลดโปรโมชันแจ้งว่าไม่พบ Python/Chrome | ติดตั้งตามหัวข้อ 5 |
| เปิดโปรแกรมแล้วไม่เห็นข้อมูลเดิมหลังย้ายเครื่อง | ใช้ "กู้คืนข้อมูล" จากไฟล์สำรอง (หัวข้อ 4) |

---

*Amway CCS Tracker — ติดตั้งแบบ per‑user, ไม่ต้องใช้สิทธิ์ admin, ข้อมูลเก็บในเครื่อง*
